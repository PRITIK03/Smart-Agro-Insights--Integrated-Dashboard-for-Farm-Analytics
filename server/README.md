# Smart Agro Insights Backend

This is the backend API for the Smart Agro Insights dashboard, built with Node.js and Express.

## Getting Started

1. Install dependencies:
   ```
   cd server
   npm install
   ```
2. Start the server:
   ```
   npm start
   ```
   Or for development with auto-reload:
   ```
   npm run dev
   ```

The server will run on port 5000 by default.

## API Endpoints
- `GET /api/health` — Health check endpoint.

Add more endpoints in `index.js` as needed.
